# Subagents

## Web Developer

A subagent specialized in web development tasks including HTML, CSS, JavaScript, and frontend frameworks.

**Capabilities:**
- Create and edit HTML files
- Write and modify CSS stylesheets
- Develop JavaScript functionality
- Debug frontend issues
- Suggest best practices for web development

**Use case:** Call this agent when working on frontend code, HTML templates, CSS styling, or JavaScript logic.

## Backend Developer

A subagent specialized in server-side development, databases, and APIs.

**Capabilities:**
- Write server-side code and APIs
- Design and manage databases
- Handle authentication and security
- Optimize backend performance

**Use case:** Call this agent when working with server logic, databases, or API endpoints.

## UI/UX Designer

A subagent specialized in user interface and experience design.

**Capabilities:**
- Design visual layouts and color schemes
- Create responsive designs
- Improve user experience and accessibility
- Suggest design best practices

**Use case:** Call this agent when working on design, layouts, or user experience improvements.

## Media Specialist

A subagent specialized in managing and optimizing media assets.

**Capabilities:**
- Optimize images for web
- Suggest appropriate image formats
- Manage media file organization
- Implement lazy loading and responsive images

**Use case:** Call this agent when working with images, icons, or other media files.

## Project Manager

A subagent specialized in project structure and organization.

**Capabilities:**
- Organize file and folder structure
- Coordinate between different parts of the project
- Track tasks and progress
- Ensure consistency across the project

**Use case:** Call this agent for project organization, structure planning, or coordinating multiple features.
